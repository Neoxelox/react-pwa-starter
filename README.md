# react-pwa-starter

Full **React** TypeScript starter with automatic bundling into a **progressive web app** featuring:

- React
- Vite
- TypeScript
- PWA
- SCSS
- React Router
- Autoprefixer
- HTML, SCSS and JS minification
- ESLint
- Prettier
- Stylelint
- WASM
- Yarn

## develop

**`yarn dev`**

## lint

**`yarn lint`**

## format

**`yarn format`**

## build

**`yarn build`**
